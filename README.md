# ASIF-CD
This repository contains the code for the paper "An Adaptive Spatial-Frequency Interaction Fusion Framework with Competitive Distillation Strategy for Optical-SAR Object Detection".

Our code is based on the implementation of the [YOLOv11] and the details can be found there.

This paper has been published in *IEEE Journal of Selected Topics in Applied Earth Observations and Remote Sensing*, 2026. If you find our paper useful for your research, please cite us.
